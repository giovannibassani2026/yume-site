# Site YUME

Site independente da YUME — tiny houses/módulos de luxo, móveis modulares e projetos ligados a Airbnb.

## Isolamento

Este site não compartilha arquivos com o projeto Vista Temporada (`negocios/airbnb-studios/site-vista-temporada`).

## Assets esperados

Colocar na pasta `assets/`:

- `Home_inicial.png` — imagem de impacto da home/hero.
- `WhatsApp Video 2026-02-26 at 11.56.58.mp4` — vídeo do módulo.
- `yume-studio-planta.png` — planta Yume Studio 22m².
- `yume-suite-planta.png` — planta Yume Suite 27m².

## Rodar localmente

```bash
python3 -m http.server 8089
```

Abrir: `http://127.0.0.1:8089`
